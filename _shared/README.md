# _shared — 공용 에셋

모든 서브도메인에서 `/_shared/파일명` 으로 접근되는 파일들.
(워커가 이 경로만 사이트 프리픽스 없이 서빙한다)

## share.js — 공유 버튼

토이의 `</body>` 직전에 한 줄:

```html
<script defer src="/_shared/share.js"></script>
```

- 우하단에 반투명 "📤 공유" 버튼이 뜬다
- 모바일: OS 공유 시트 (카카오톡·문자 등 포함) / 데스크톱: 링크 복사 + 토스트
- 전체 화면이 클릭 영역인 토이에도 안전 (이벤트 전파 차단)

### 자랑 문구

토이 스크립트에서 전역 훅 하나만 정의하면 공유 내용에 문구가 실린다:

```js
window.blShareText = () => `내 반응속도는 ${best}ms! ⚡ 도전해보세요`;
// 함수 대신 문자열도 가능. 없으면 링크만 공유.
```

클립보드 복사 시에는 `문구\nURL` 형태로 복사된다.

### 나중에: 카카오톡 전용 버튼

공유 시트 안 거치고 바로 카톡을 띄우려면 카카오 개발자 앱 등록 +
JavaScript 키가 필요하다. 키가 생기면 share.js에 SDK 연동을 추가할 것.
