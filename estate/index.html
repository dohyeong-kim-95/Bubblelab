데이터 준비가 필요합니다

국토부 실거래가 API는 해외 IP를 차단해 Cloudflare 서버가 직접 가져올 수 없습니다. 한국 IP인 내 컴퓨터에서 받아서 정적 파일로 커밋하는 방식을 씁니다.

  1. data.go.kr 가입 후 아파트 매매 실거래가 자료, 아파트 전월세 실거래가 자료 두 API를 활용신청 (자동승인)
  2. 마이페이지에서 일반 인증키 (Decoding) 복사
  3. 리포 루트 .dev.vars 파일에 MOLIT_SERVICE_KEY=키 한 줄 추가 (.gitignore에 있어 커밋되지 않음)
  4. node _infra/estate-import.mjs 실행 → estate/data/에 월별 JSON 생성
  5. estate/data를 커밋·푸시 → 배포 후 이 페이지 새로고침. 새 신고분을 반영하려면 가끔 다시 돌려서 커밋

로컬 개발(wrangler dev)은 한국 IP라 .dev.vars의 키만으로 프록시가 바로 동작합니다 — 커밋 없이도 화면을 볼 수 있어요.
