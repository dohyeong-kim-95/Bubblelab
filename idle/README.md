# idle — Bubble Pop Idle

<https://idle.bubblelab.dev>에서 운영하는 7일 유한형 방치 게임입니다. 매주 월요일
09:00 KST에 새 시즌이 시작되며, 다음 월요일 같은 시각에 생산을 멈추고 기록을
확정합니다.

## 경로와 파일

| 경로 | 내용 |
| --- | --- |
| `bubble-pop/` | 게임 UI, 저장·오프라인 계산·최종 기록 제출 |
| `hall-of-fame/` | 시즌별 Bubble Pop Idle 우승 기록과 규칙 |
| `index.html` | idle 카테고리 홈 |
| `docs/progression-research.md` | 방치 게임 진행 설계 리서치 |
| `../_infra/idle-balance.mjs` | 밸런스·콘텐츠 소진 시뮬레이터 |

## 게임 루프

```text
버블 직접 터뜨리기
→ 생산기 구매와 초당 생산 증가
→ 최대 24시간 오프라인 보상
→ 모든 생산기 보유 후 버블 압축 해금
→ 압력 빌드 선택
→ 7일 종료와 명예의 전당 제출
```

핵심 규칙:

- 참가 전 시간의 생산량은 소급하지 않습니다.
- 첫 버블 막대는 무료이며 생산기 구매마다 기본 생산과 해당 생산기 효율이 오릅니다.
- 생산기 가격은 기하급수적으로 오르고, 25개 단위마다 생산량이 2배가 됩니다.
- 구매 단위는 1·10·100·최대이며 길게 눌러 연속 구매할 수 있습니다.
- 모든 생산기를 하나 이상 보유하면 `버블 압축`이 열립니다.
- 압력은 자동 생산, 직접 보상, 오프라인 효율, 압력 생산에 나누어 투자합니다.
- 누적 500·50K·5M 버블에서 진주·황금·오로라 버블이 해금됩니다.
- 프레스티지, 결제, 광고, 랜덤 상자, 연속 접속 보상은 없습니다.

## 저장과 기록

진행 상태는 버전이 있는 단일 `localStorage` 값으로 저장합니다. 주간 기록은
`/_records`의 `bubble-pop-idle` 게임 키로 제출되며, 일반 Slop 기록과 달리 최대
52개 시즌 역사를 보존해 `hall-of-fame/`에 표시합니다.

기록 시스템은 친구끼리 즐기는 양심 기록입니다. 로컬 시각 변경이나 개발자 도구
조작을 막는 경쟁형 부정행위 방지는 현재 범위에 포함하지 않습니다.
