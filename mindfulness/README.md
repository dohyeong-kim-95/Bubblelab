# mindfulness — 짧은 웹 명상

<https://mindfulness.bubblelab.dev>에서 제공하는 짧은 호흡·소리·생각 알아차림
실험입니다. 현재 공개 상태는 **Beta**이며 의료·심리 치료를 대신하지 않습니다.

| 경로 | 경험 | 데이터 처리 |
| --- | --- | --- |
| `breathe/` | 호흡을 따라 잠깐 멈추기 | 입력·계정 없음 |
| `sound-bubble/` | 소리방울과 함께 듣기 | 입력·계정 없음 |
| `thought-bubble/` | 생각을 버블에 담아 흘려보내기 | 입력 문구를 DOM에만 표시하고 저장·전송하지 않음 |
| `listen/` | Upward Drift 듣기 | `assets.bubblelab.dev`의 정적 음원 재생 |

## 개인정보와 네트워크

- 생각방울의 문구는 `textContent`로 브라우저 화면에만 표시되며 서버, 쿠키,
  `localStorage`에 저장하지 않습니다.
- 회원가입, 이메일, 마이크 입력은 사용하지 않습니다.
- Bubblelab 공통 집계가 페이지 경로, 익명 브라우저 ID와 활성 체류시간을 기록할 수
  있지만 사용자가 입력한 생각이나 명상 내용은 집계 요청에 포함하지 않습니다.
- 음악·미리보기는 Bubblelab의 같은 사이트 범위인 `assets.bubblelab.dev`에서만
  불러옵니다.

새 명상 카드는 민감한 자유 입력을 서버로 보내지 않는 것을 기본으로 하고, 전송이
필요해지는 경우 Privacy 문서와 명시적 동의를 먼저 마련해야 합니다.
