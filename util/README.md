# util — 일상 도구

카테고리 상태: **Beta**. 도구별 상태와 데이터 처리가 서로 다릅니다.

<https://util.bubblelab.dev>에서 제공하는 작은 생활 도구입니다. 사이트 루트 카드
페이지는 빌드 시 자동 생성됩니다.

| 경로 | 상태 | 도구 | 데이터 처리 |
| --- | --- | --- | --- |
| `calendar/` | Beta | 달력 이동과 공유 | 서버 저장 없음 |
| `convert/` | Beta | 이미지 JPG/PNG 변환 | 이미지가 브라우저 안에서만 처리됨 |
| `fortune/` | Beta | 명식과 오늘의 운세 | 요청 순간 계산, 서버 저장 없음 |
| `photo/` | Beta | 카메라 네컷사진 | 이미지가 브라우저 안에서만 처리됨 |
| `planner/` | Archived | 개인 코드 월간 플래너 | API 비활성·카드 비노출 |

## 운세

`fortune/`은 `/_fortune/chart`를 호출합니다. Worker가 `KASI_SERVICE_KEY`를 이용해
양력·음력 날짜를 변환하며 키는 브라우저에 노출하지 않습니다. 생년월일시는 서버에
저장하지 않고 요청 순간에만 처리합니다. 입력값은 편의를 위해 해당 브라우저의
`localStorage`에 저장될 수 있습니다.

`/_fortune/chart`는 동일 출처 JSON 요청만 받고 본문 크기와 분당 호출 수를 제한합니다.
KASI 키는 Worker secret에만 있으며 응답이나 정적 JavaScript에 포함되지 않습니다.

결과에는 사주 명식, 오늘의 종합 문구, 재물운·직업운·연애운 상태와 설명이
포함됩니다. 공유 카드는 분야별 운세와 `util.bubblelab.dev/fortune` 주소를 넣어
이미지로 생성합니다.

## 네컷사진과 달력

`photo/`는 사용자가 허용한 카메라 영상을 브라우저 Canvas에서 조합합니다. 촬영
이미지나 완성 이미지를 Bubblelab 서버로 업로드하지 않습니다. 보안 헤더는 카메라를
같은 출처에만 허용하고 마이크·위치 권한은 차단합니다. `calendar/`도 서버 계정이나
일정 저장소를 사용하지 않습니다.

## Planner

플래너는 숫자 6자리와 영문 2자로 이루어진 개인 코드를 사용합니다. 같은 코드를
입력한 PC 편집 화면과 모바일 조회 화면이 `PlannerDO`의 같은 데이터를 봅니다.
서버는 KST 현재 달만 유지하며 코드는 복구할 수 없습니다.

운영에는 `PLANNER_SESSION_SECRET` 또는 대체 가능한 관리자 세션 secret이 필요합니다.
브라우저가 직접 Durable Object를 호출하지 않고 `/_planner/*` 세션 API를 사용합니다.
공개 보안 정비 중에는 코드만 아는 사람을 소유자로 보는 현재 인증 방식을 외부에
노출하지 않도록 카드 목록에서 숨기고 `ENABLE_PLANNER=false`로 API를 닫아 둡니다.
다시 열려면 코드 추측 방지뿐 아니라 코드 소유권, 복구, 분실·공유, 삭제 요청 정책을
먼저 정해야 합니다.
