# www — Bubblelab 랜딩

상태: **Beta**. 공개 프로젝트의 상태와 시작점을 안내하는 메인 화면입니다.

`www/index.html`은 <https://bubblelab.dev>와 <https://www.bubblelab.dev>에 제공되는
가벼운 정적 랜딩 페이지입니다.

현재 랜딩은 다음 카테고리를 소개합니다.

- `slop`: 짧은 게임과 실험
- `mindfulness`: 짧은 호흡·소리·생각 알아차림
- `games`: 현재 실시간 서버 정비로 Archived 상태
- `util`: 일상 도구
- `idle`: 7일 방치형 게임

`assets`, `invest`, `admin`도 독립 서브도메인으로 배포되지만 랜딩 목록에는 아직
노출하지 않습니다. 새 공개 카테고리를 랜딩에 보여주려면 `www/index.html`의 목록을
직접 수정해야 합니다. 빌드 스크립트가 이 페이지의 링크를 자동 갱신하지는 않습니다.

현재 `www/index.html`의 문구와 링크는 초기 랜딩이므로, 공개 준비의 다음 단계에서
대표 문장, 추천 실험, 상태가 표시된 카테고리, 최근 업데이트, 운영자 소개 구조로
개편해야 합니다. Archived 카테고리를 링크할 때는 정비 중임을 화면에도 표시합니다.

랜딩 자체는 자유 입력을 받지 않습니다. 일반 공개 페이지와 동일하게 익명 브라우저
ID 기반 방문 집계가 적용될 수 있으며 관리자·Planner·실시간 데이터는 노출하지
않습니다.
