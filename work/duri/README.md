# duri — 둘이서 쓰는 기록 시스템 (진행 미리보기)

상태: **진행 중**. work.bubblelab.dev/duri 에서 비밀번호 입력 후 확인.

[Duri](https://github.com/dohyeong-kim-95/Duri)는 두 사람의 대화·사진·일정을
자동으로 남기고 하나의 타임라인으로 평생 보존하는 개인 라이프 시스템입니다.
실서비스는 별도 리포·별도 배포(<https://duri.bubblelab.dev>, 초대 코드 + 기기
등록으로 두 사람만 접근)이고, 이 폴더는 **진행 상황 공유용 정적 미리보기 한 장**입니다.

- 대화·사진 백업이 목적이라 bubblelab의 "토이 · 외부 서비스 금지" 원칙과는
  결이 다릅니다. 그래서 랜딩·풀다운에 노출되지 않는 비공개 `work` 서브도메인의
  외주 프로젝트로 두었습니다(패스워드 게이트 · noindex · 통계 제외).
- 자체 완결: `_shared` 등 리포 공용 리소스에 의존하지 않고 이 폴더의
  `index.html` 하나만으로 구동됩니다. 서버에 아무것도 저장하지 않습니다.
- 페이지 내용(개념·MVP·Phase 4 상태·기술 스택)은 Duri 리포 README를 옮긴
  것입니다. 진행 단계가 바뀌면 `index.html`의 `.status` 블록을 갱신합니다.
- 실제 데이터·인증·저장 로직은 Duri 리포에 있으며 이 미리보기에는 포함되지
  않습니다. 인계·서비스 종료 시 이 폴더를 삭제합니다.
