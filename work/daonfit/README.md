# daonfit — 다온핏 브랜드 홈페이지 (외주 시안)

상태: **작업 중**. work.bubblelab.dev/daonfit 에서 비밀번호 입력 후 확인.

시안 두 가지를 나란히 운영합니다 — 루트는 선택 페이지이고, `v1/`은 따뜻한
크림 톤(e1b023b 시점), `v2/`는 시네마틱 브라운(Shopifi 문법 절충)입니다.
두 버전은 같은 `/_workqna/daonfit` Q&A 데이터를 공유합니다. 방향이 확정되면
남는 버전 하나를 루트로 승격하고 나머지는 삭제합니다.

3D프린팅 생활 맞춤 소품 브랜드 "다온핏"(스마트스토어 `samyong_2`,
인스타그램 `@daonfit`)의 자체 홈페이지 시안입니다. 구매는 스마트스토어로
연결하며 이 사이트는 서버에 아무것도 저장하지 않습니다.

- `_work_assets/`의 로고·상품 사진 권리는 다온핏에 있습니다. 무단 사용 금지.
  현재 상품 사진은 스토어 스크린샷 기반 임시본으로, 인계 전 원본으로 교체합니다.
- 브랜드 슬로건: "좋은 것들이 모여, 너에게 꼭 맞게"
- 문의 Q&A: `qna.html`(고객)과 `admin/`(관리자)이 `/_workqna/daonfit` API를
  사용합니다. work 게이트 세션이 있어야 읽기·쓰기가 가능하며, 문의는 닉네임과
  내용만 받고(개인정보 미기재 안내) `WorkQnaDO`에 프로젝트별로 저장됩니다.
  주의: 이 API는 bubblelab Worker에 있으므로 정적 인계 시에는 별도 백엔드로
  옮기거나 스마트스토어 문의로 대체해야 합니다.
- 인계 시 이 폴더를 통째로 전달하고 리포에서 삭제합니다.
