# invest — Bubble Invest

상태: **Experiment**. Bubblelab 대표 공개 프로젝트나 실제 금융 서비스가 아닙니다.

<https://invest.bubblelab.dev>에서 제공하는 Bloomberg Terminal풍 개인 투자 화면
프로토타입입니다. PC에서는 넓은 HTS 레이아웃, 모바일에서는 하단 5개 탭의 MTS
레이아웃으로 바뀝니다.

현재 기능:

- 미국 주식·ETF 11종 검색과 관심종목 관리
- 1D–5Y 범위의 캔들·라인 차트 전환
- 포인터·터치 이동에 따른 OHLC/가격 툴팁
- 뉴스·기업정보·재무·매크로 탭
- 모의 매수·매도와 로컬 포트폴리오 손익
- 모바일 홈·관심·차트·뉴스·자산 탭과 로딩 스켈레톤
- 선택 종목, 관심종목, 차트 유형, 모의 포트폴리오를 `localStorage`에 저장

## 중요한 제한

모든 시세, 차트, 뉴스, 재무와 거시 지표는 UI 검증용 고정·생성 데모 데이터입니다.
실시간 데이터나 TradingView, 증권사 주문 API와 연결되어 있지 않으며 투자 판단이나
실제 주문에 사용하면 안 됩니다.

관심종목과 모의 포트폴리오는 해당 브라우저의 `localStorage`에만 저장되며 서버에
전송하지 않습니다. Bubblelab 공통 익명 방문·활성 체류 집계에는 페이지 경로와
체류시간만 포함되고 종목 선택, 보유 수량, 모의 손익은 포함되지 않습니다.

실시간 시세·추천·목표가·실제 주문 기능을 붙이는 변경은 현재 Experiment 범위를
벗어나므로 투자 면책, 데이터 제공자 약관, 오류·지연 표시를 별도로 검토해야 합니다.

## 파일

- `index.html`: 화면 구조와 접근 가능한 버튼·탭
- `styles.css`: 데스크톱·모바일 반응형 레이아웃, 차트 제스처 영역, 스켈레톤
- `app.js`: 데모 데이터, 차트, 검색, 주문과 로컬 상태

외부 빌드 과정이나 런타임 의존성 없이 정적 파일로 배포됩니다.
